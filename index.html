  • Компресор для векторних зображеннь SVG - https://svgomg.net/
  • Lazy loading (githab з кодом для ледачого завантаження) - https://github.com/verlok/vanilla-lazyload
  • Однепіксельне зображення (для заповнення src для lazy loading) - data:image/gif;base64,R0lGODlhBAADAIAAAP///wAAACH5BAEAAAEALAAAAAAEAAMAAAIDjI9WADs=
  • Тег img з однопіксельним зображенням -
